#  Movie Recommender System

A content-based movie recommendation system using cosine similarity.

##  Overview
This project recommends movies similar to a given movie by computing cosine similarity between content-based feature vectors.

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

##  Features
- Content-based filtering
- Cosine similarity for recommendation
- Visualization of similarity scores

##  Future Improvements
- Convert into a Streamlit web app
- Add user-based collaborative filtering
